Welcome to Maximus78!

You'll need to complete the following steps first to enjoy the full experience of Maximus78 with all of the associated artwork, manuals, help files, and screenshots. 

1. Download the A7800 Windows Package
 Unzip it to the folder of your choice.
https://github.com/7800-devtools/a7800/releases/download/v5.2/a7800-win-v5.2.zip

This file includes the A7800 executable required for Maximus78.

2. Download the A7800 Optional Files
Unzip this package to the same root folder location as A7800. 
https://archive.org/download/a-7800-optional-files-20220628/A7800_Optional_Files_20220628.zip

This file includes two folders, "bios" and "roms".  While optional, you will want to download this archive to use all of the features of Maximus78 and A7800.  Note that the roms folder contains an older version of Trebor's 7800 ROM pack, so you'll want to dowload the latest version in the next step.

3. Download Trebor's 7800 ROM ProPack
Unzip it anywhere you'd like, but I'd suggest the roms folder in your A7800 Directory.
http://7800.8bitdev.org/index.php/Trebors_7800_ROM_PROPack

This file includes the latest compilation of Atari 7800 Homebrews, Hacks, Demos, Prototypes, and all retail releases in both NTSC and PAL formats.  You can safely overwrite the contents of the ROMs folder with this file if you have an older version of Trebor's ROM pack. It's important to note that all of the screenshots, manuals, and box art are tied to the exact file names in Trebor's ROM Pack, it's important that you do not rename an of the ROM files for Maximus78 to function properly.

4. Download the Game Information zip file (includes the Manauls and screenshots) for Maximus78.
You can unzip it anywhere, however I would suggest creating a "gameinfo" subfolder in your main A7800 Directory.
You'll need to download all of the files, they are a split archive due to the 25MB file limit on GitHub.
Maximus78_Manuals.zip
Maximus78_Manuals.z01
Maximus78_Manuals.z02
Maximus78_Manuals.z03
Maximus78_Manuals.z04
Maximus78_Manuals.z05
Maximus78_Manuals.z06
Maximus78_Manuals.z07
Maximus78_Manuals.z08

The files includes all of the screenshots and manuals for Maximus78. The manuals archive will be periodically updated as it does not contain screenshots and/or manuals for every game in Trebor's ROM pack.

The files included in the Maximus78 GitHub repository at https://github.com/AtariusMaximus/Maximus78

5. Download the Box Art zip file for Maximus78.
You can unzip it anywhere, however I would suggest creating a "boxart" subfolder in your main A7800 Directory.

This file includes all of the box art for Maximus78. The manuals archive will be periodically updated as it does not contain screenshots and/or manuals for every game in Trebor's ROM pack.
Maximus78_BoxArt.zip

This file is included in the Maximus78 GitHub repository https://github.com/AtariusMaximus/Maximus78
